# Aula 01 - Lógica de Programação

## O que é algoritmo
Algoritmo é uma estrutura de passos feitos para resolver determinado problema.

## Entrada, processamento e saída

Entrada: dados que são enviados ao sistema.

Processamento: a maneira como o sistema lida com esses dados.

Saída: o resultado final.

## Exemplo do dia a dia

Escovar os dentes:

1. Pegar a escova
2. Colocar pasta de dente
3. Escovar os dentes
4. Enxaguar a boca

## Palavras-chave

Algoritmo: sequência de passos para resolver algo.

Entrada: informação recebida.

Processamento: tratamento das informações.

Saída: resultado produzido.

Lógica: forma organizada de pensar para resolver problemas.
